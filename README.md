# 🌈 Rainbow Speed Test - React Edition

A high-performance, vibrant, and unique internet speed test application built with **React**, **Tailwind CSS v4**, and **Framer Motion**.

![Speed Test Preview](https://raw.githubusercontent.com/satetapongsa/archtest2/main/preview.png)

## ✨ Features

- **Horizontal Half-Circle Gauge**: A custom-designed, horizontal half-circle speedometer with real-time rainbow progress.
- **Vibrant Rainbow Aesthetic**: Unique design system featuring glowing rainbow gradients on borders, text, and graphs.
- **Real-Time Data Visualization**: Dynamic rainbow graphs for both Download and Upload speeds.
- **Comprehensive Metrics**: Measures Ping, Jitter, Download, and Upload speeds with high accuracy simulations.
- **Premium Dashboard Layout**: A professional, balanced dashboard organized into clear functional zones.
- **Interactive Elements**: Glassmorphism cards, glowing buttons, and smooth transition effects.
- **Social Integration**: Easily copy results or share on Twitter and Facebook.

## 🚀 Tech Stack

- **Framework**: [React](https://reactjs.org/) (via [Vite](https://vitejs.dev/))
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: Custom SVG paths for high-performance and unique styling.

## 🛠️ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/satetapongsa/archtest2.git
   cd archtest2
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.

## 🎨 Design Philosophy

The **Rainbow Speed Test** was designed to move away from the generic blue/white speed test interfaces. By combining dark mode with vibrant neon-rainbow accents, the application provides a "Wow" factor while maintaining professional functionality. The use of custom SVGs ensures that every pixel of the rainbow experience is sharp and performance-oriented.
